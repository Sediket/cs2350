What is the difference between TCP/IP and HTTP?

TCP/IP stands for Transmission Control Protocol/Internet Protocol. TCP/IP is the most popular form of modern networking between capable devices, for example computers or phones when accessing the internet; TCP/IP establishes a suite of protocols and rules for this networking.

Define the term: URL?

URL stands for Uniform Resource Locator and are commonly used to location web sites or ftp or email addresses, but can be used for referencing pretty much anything. An example would be: http://www.google.com/index.html which is connecting to the server at www.google.com and the filename index.html.

What is the Client/Server Model?

The client/server model is the basis for sharing of information on networks. Typically the server will be hosting information, information that will be available for clients to access. This communication is initiated by clients when they request for this information from the server, then is shared.

What two things does the browser do?

  1. A web browser downloads a webpage from a server and then
  2. Displays its contents, rendered appropriatley, for the user.

Terms:

  • HTML
    • Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
  • CSS
    • CSS is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed.
  • JavaScript
    • An object-oriented computer programming language commonly used to create interactive effects within web browsers.
